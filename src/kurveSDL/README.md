# kurveSDL
This is the "predecessor" of IPCurve: a version of *Achtung, die Kurve!* implemented in FreePascal using SDL libraries. It dates back to ~2005. Finally, when it was not possible to add network play as the engine was not written with this in mind, I went for a re-write in C++, called **IPCurve**, based on a client-server architecture from the start.

If you just want to play the game, play IPCurve. This is more of a historical artifact.
